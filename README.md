# BART vs T5 Scientific Paper Summarization

## Project Overview
This project will comapare two pretrained AI summarization models, BART and T5, on scientific research papers. The goal is to evaluate which model will produce more accurate, readable, and useful summaries of scientific text.

## Models Used
- facebook/bart-larg-cnn
- google/flan-t5-base

## Repository Structure
- data/ - Dataset information and preprocessing scripts
- models/ - Model implementation and configurations
- experiments/ - Experimental notebooks and results
- docs/ - Documentation and reports

## How to Run
Open the the notebooks in the experiments/ folder using Google Colab

## Evaluation Metrics
- ROGUE Score
- BERTScore
